# DMART Sales Analysis Dashboard

## Project Overview

This project is a Power BI dashboard that provides insights into DMART sales data. It helps analyze revenue, customer behavior, order trends, and marketing performance.

---

## Key Features

* Total Revenue: ₹17M+
* Total Orders: 25K+
* Unique Customers: 24K+
* Cancellation tracking
* Monthly and quarterly sales trends
* City-wise sales analysis
* Marketing performance insights
* Payment method distribution

---

## Dashboard Insights

### Sales Performance

* Revenue shows growth across quarters
* Highest sales observed in Q4 (~4.37M)

### Customer Insights

* Male customers are higher than female
* Majority of customers fall between age 20–60

### Order Status Analysis

* Delivered orders are the highest
* Some cancellations and returns observed

### Category Performance

* Local category contributes the highest revenue
* Branded and imported categories follow

### Marketing Insights

* Instagram and Facebook generate the most revenue
* Marketing channels impact overall sales significantly
